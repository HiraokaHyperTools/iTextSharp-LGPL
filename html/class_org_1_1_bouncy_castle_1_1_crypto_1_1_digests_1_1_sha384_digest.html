Detailed Description

Draft FIPS 180-2 implementation of SHA-384. Note: As this is based on a draft this implementation is subject to change.

        block  word  digest
SHA-1   512    32    160
SHA-256 512    32    256
SHA-384 1024   64    384
SHA-512 1024   64    512
