Detailed Description

The RtfParserState contains the state information for the parser. The current state object is pushed/popped in a stack when a group change is made.

When an open group is encountered, the current state is copied and then pushed on the top of the stack When a close group is encountered, the current state is overwritten with the popped value from the top of the stack
